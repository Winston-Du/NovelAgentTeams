# 市集开局（试读样例）

落日沉进槐城的雾，市集尽头的货栈挂着一盏裂灯。有人把灯火吹灭，有人把价格抬高；陆商曜拨动算盘，珠声在掌心里轻轻滚。

"老规矩，摊位费三成，保护费两成，过关再收一成。"黑商周桓笑得像掰断脆骨，"签了，明天你还开门。"

木九公抱着破账本咳了一声。陆商曜抬眸，语气平稳："可以签，但按城里常用的合同样式来。你收'保护'，遇到非买卖的打砸抢，要负责。若出现仓毁货损，以当月收取费用十倍赔付；若你的人参与此事，合同当场失效，已收费用全退并公开道歉。"

摊位间有人嘘，有人笑。周桓指节在桌上敲得沉闷："你拿几句漂亮话当盾牌？"

"不是。"陆商曜把纸推过去，"是拿清楚的字句当尺子。"

风从巷口压进来，后墙忽然响动，十几条影子翻越而入。周桓冷笑："'非买卖'来了。"

陆商曜起身，把账本扣上："铁阙。"

暗处走出两名壮汉，抬起厚木盾，把门口一挡。门闩落下，门外被挤开的脚步立刻乱了。有人抡棒砸木盾，回声在窄巷里碰撞；有人想冲窗，被横木顶住，只能骂。

"保护不是嘴上说。"陆商曜的声音很平，"站在我们这边，才叫保护。"

影子退了几步，巷口有人吹了口哨，脚步散开。铁阙把门闩提起，木九公把桌面上的纸整理好。

陆商曜把合同重新推到周桓面前："现在可以签。你不认也行，今天这些见到的人都能作证，城里也会管这类事。"

周桓的笑意裂开了一寸。他抬眼看雾，又看了一眼那盏裂灯。

袖口里，一枚旧印记似乎微微发热。陆商曜没有去看，他只把算盘珠轻轻拨到一侧。

灯火亮了一点，像有人在风里护住了火心。珠子叮的一下，落在账面上，也落在他今天划下的一条线旁边。

这一条线，之后会越划越清楚。

## 技巧总结

### 1. Show, don't tell 示例
- ❌ "陆商曜很镇定"
- ✅ "陆商曜拨动算盘，珠声在掌心里轻轻滚"

### 2. 环境渲染技巧
- 用具体的细节建立氛围："落日沉进槐城的雾，市集尽头的货栈挂着一盏裂灯"
- 不直接说"环境压抑"，而是通过"灯火摇晃""巷口风声"来暗示

### 3. 对话张力设计
- 每句对话都推动剧情或展现人物
- 周桓的粗鲁 vs 陆商曜的精确，用语言本身表现冲突
- "你拿几句漂亮话当盾牌？" vs "不是。是拿清楚的字句当尺子。" ← 一句话碾压

### 4. 动作表现人物
- 木九公：抱着破账本咳嗽 → 暗示警告和专业性
- 陆商曜：拨动算盘 → 表现冷静思考
- 周桓：拳头敲桌子 → 表现暴力倾向

### 5. 节奏控制
- 长句铺陈环境：开场的环境描写
- 短句点睛：对话简洁有力
- 紧张时用短句："影子退了几步，巷口有人吹了口哨，脚步散开。"

### 6. 伏笔埋设
- "袖口里，一枚旧印记似乎微微发热" ← 自然且不突兀
- "这一条线，之后会越划越清楚" ← 暗示主角的成长轨迹
